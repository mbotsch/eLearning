# eLearning

Ein Beispielvortrag, der die Features von interaktiven HTML-Folien aufzeigt.

Folien werden im Markdown-Textformat geschrieben und mit der Bibliothek [pandoc](https://github.com/hakimel/reveal.js/) in das HTML-Folienformat von [reveal.js](https://github.com/hakimel/reveal.js/) Ã¼bersetzt. Unser Tool [decker](https://gitlab2.informatik.uni-wuerzburg.de/decker/decker) vereinfacht diesen Prozess und erweitert pandoc und reveal in vielfacher Hinsicht. Einfach mit `decker server` die Folien generieren und im Webbrowser `http://localhost:8888/` aufrufen.
